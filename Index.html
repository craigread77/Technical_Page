Details

The details tag specified additional details that the user can view or hide on demand. The content of a details element should not be visisble unless the open attribute is set.


<details>
    <summary>Copyright 1999-2018.</summary>
    <p> - by Refsnes Data. All Rights Reserved.</p>
    <p>All content and graphics on this web site are the property of the company Refsnes Data.</p>
</details>

As you can see from the example above, details can have pretty much anything nested inside of it.
Some good reasons to use the details tag are:


  • Increased accessibility without hindering the normal page experience.
  • Good habit to get used to new features.
  • Using new features is impressive.
  • Users can see more info without crowding the page.
  • Integrates well with javascript to show details in a new box upon click.

Figcaption

The figcaption tag defines a caption for a figure element.

The figcaption element can be placed as the first or last child of the figure element.


<figure>
    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>
